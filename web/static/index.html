安全檢查

Swap 報價

輸入從哪個代幣、數量、換成哪個代幣，取得預估可換數量。

同鏈 Swap（Calldata + 錢包簽名）

僅 EVM 同鏈（例如 Base 上 USDC→WETH）。流程：① 報價拿 market → ② 取得 calldata → ③ 用已連接錢包送出（eth_sendTransaction），錢包會跳出確認。請先將錢包切到對應網路。若需跨鏈請用下方 Order 模式。

半自動 Swap（Order 模式）

依序：① 取得報價 → ② 建立訂單（取得待簽名資料）→ ③ 在錢包內簽名 → ④ 貼上已簽名交易並送出。本頁不持有私鑰，僅代為取得報價、建立訂單、代為廣播已簽名交易。
